# マスタリング AWS CDK

このソースコードは [マスタリング AWS CDK](https://techbookfest.org/product/wEWtvT8aRZgzAcVSnE7wS8)用のサンプルコードです。

実行前に、2章の「AWS CDK の開発環境セットアップ」と以下のbootstrapコマンドを実行してください。

```bash
cdk bootstrap aws://123456789012/ap-northeast-1
```
